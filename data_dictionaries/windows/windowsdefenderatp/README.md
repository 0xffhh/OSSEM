# Windows Defender Advanced Threat Protection Event Logs

## Data Dictionaries
| Event Name | Description |
|--------|---------|---------|
| [AlertEvents](AlertEvents.md) | Alerts on Windows Defender Security Center |
| [MachineInfo](MachineInfo.md) | Machine information, including OS information |
| [MachineNetworkInfo](MachineNetworkInfo.md) | Network properties of machines, including adapters, IP and MAC addresses, as well as connected networks and domains |
| [ProcessCreationEvents](ProcessCreationEvents.md) | Process creation and related events |
| [NetworkCommunicationEvents](NetworkCommunicationEvents.md) | Network connection and related events |
| [FileCreationEvents](FileCreationEvents.md) | File creation, modification, and other file system events |
| [RegistryEvents](RegistryEvents.md) | Creation and modification of registry entries |
| [LogonEvents](LogonEvents.md) | Login and other authentication events |
| [ImageLoadEvents](ImageLoadEvents.md) | DLL loading events |
| [MiscEvents](MiscEvents.md) | Multiple event types, such as process injection, creation of scheduled tasks, and LSASS access attempts |
