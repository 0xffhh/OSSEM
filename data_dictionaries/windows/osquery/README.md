# Windows Osquery Event Logs

## Description
Osquery schema is defined in tables by osquery engineers.

## Sub Data Sets
|events|Description|Tags|
|---|---|---|
|[file](events/file.md)|Interactive filesystem attributes and metadata.||
|[hash](events/hash.md)|Filesystem hash data.||
|[ie_extensions](events/ie_extensions.md)|Internet Explorer browser extensions.||
|[pipes](events/pipes.md)|Named and Anonymous pipes.||
|[services](events/services.md)|List all installed Windows services and their relevant data.||
