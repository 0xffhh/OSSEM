# Audit RPC Events

Audit RPC Events determines whether the operating system generates audit events when inbound remote procedure call (RPC) connections are made.

[Microsoft Source](https://docs.microsoft.com/en-us/windows/security/threat-protection/auditing/audit-rpc-events)

## Data Dictionaries

| EventId | Description | Minimum OS |
|--------|---------|-------|
| 5712 | A Remote Procedure Call (RPC) was attempted. | Windows Vista, Windows Server 2008 |
