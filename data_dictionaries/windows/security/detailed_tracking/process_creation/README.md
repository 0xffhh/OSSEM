# Audit Process Creation

Audit Process Creation determines whether the operating system generates audit events when a process is created (starts).

[Microsoft Source](https://docs.microsoft.com/en-us/windows/security/threat-protection/auditing/audit-process-creation)

## Data Dictionaries

| EventId | Description | Minimum OS |
|--------|---------|-------|
| [4688](https://github.com/Cyb3rWard0g/OSSEM/blob/master/data_dictionaries/windows/security/events/event-4688.md) | A process has exited. | Windows Vista, Windows Server 2008 |
| [4696](https://github.com/Cyb3rWard0g/OSSEM/blob/master/data_dictionaries/windows/security/events/event-4696.md) | A primary token was assigned to process. | Windows Vista, Windows Server 2008 |
