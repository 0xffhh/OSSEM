# Audit Directory Service Access

Audit Directory Service Access determines whether the operating system generates audit events when an Active Directory Domain Services (AD DS) object is accessed.

[Microsoft Source](https://docs.microsoft.com/en-us/windows/security/threat-protection/auditing/audit-directory-service-access)

## Data Dictionaries

| EventId | Description | Minimum OS |
|--------|---------|-------|
| [4662](https://github.com/Cyb3rWard0g/OSSEM/blob/master/data_dictionaries/windows/security/events/event-4662.md) | An operation was performed on an object. | Windows Vista, Windows Server 2008 |
| [4661](https://github.com/Cyb3rWard0g/OSSEM/blob/master/data_dictionaries/windows/security/events/event-4661.md) | A handle to an object was requested. | Windows Vista, Windows Server 2008 |
