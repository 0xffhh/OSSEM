# Audit Audit Policy Change

Audit Audit Policy Change determines whether the operating system generates audit events when changes are made to audit policy.

[Microsoft Source](https://docs.microsoft.com/en-us/windows/security/threat-protection/auditing/audit-audit-policy-change)

## Data Dictionaries

| EventId | Description | Minimum OS |
|--------|---------|-------|
| 4715 | The audit policy (SACL) on an object was changed. | Windows Vista, Windows Server 2008 |
| 4719 | System audit policy was changed. | Windows Vista, Windows Server 2008 |
| 4817 | Auditing settings on an object were changed. | Windows 7, Windows Server 2008 R2 |
| 4902 | The Per-user audit policy table was created. | Windows Vista, Windows Server 2008 |
| 4904 | An attempt was made to register a security event source. | Windows Vista, Windows Server 2008 |
| 4905 | An attempt was made to unregister a security event source. | Windows Vista, Windows Server 2008 |
| 4906 | The CrashOnAuditFail value has changed. | Windows Vista, Windows Server 2008 |
| 4907 | Auditing settings on object were changed. | Windows Vista, Windows Server 2008 |
| 4908 | Special Groups Logon table modified. | Windows Vista, Windows Server 2008 |
| 4912 | Per User Audit Policy was changed. | Windows Vista, Windows Server 2008 |
