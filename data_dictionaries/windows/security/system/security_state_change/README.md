# Audit Security State Change

Audit Security State Change contains Windows startup, recovery, and shutdown events, and information about changes in system time.

[Microsoft Source](https://docs.microsoft.com/en-us/windows/security/threat-protection/auditing/audit-security-state-change)

## Data Dictionaries

| EventId | Description | Minimum OS |
|--------|---------|-------|
| 4608 | Windows is starting up. | |
| 4616 | The system time was changed. | |
| 4621 | Administrator recovered system from CrashOnAuditFail. | |
