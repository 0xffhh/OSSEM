# Date Object

## Data Fields

| field name | field type | description |
|--------|---------|-------|
| date_network_start | date | Specifies the date/time the network traffic was initiated |
| date_network_end | date | Specifies the date/time the network traffic ended |
| date_creation | date | Specifies the date/time the file was created |
| date_modified | date | Specifies the date/time the file was last written to/modified |
| date_accessed | date | Specifies the date/time the file was last accessed |
| date_alert_triggered | string | Specifies the date/time an alert triggered |
