# Port Object

## Data Fields

| field name | field type | description |
|--------|---------|-------|
| port_src_name | string | source port name (http,netbios-dgm, etc) |
| port_src_number | integer | port number from where the network connection originated |
| port_dst_name | string | destination port name (http,netbios-dgm, etc) |
| port_dst_number | string | port number of the destination in a network connection event |
