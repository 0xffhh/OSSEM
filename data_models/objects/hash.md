# Hash Object

## Data Fields

| field name | field type | description |
|--------|---------|-------|
| hash_md5 | string | MD5 hash of the image/binary/file |
| hash_sha1 | string | SHA1 hash of the image/binary/file |
| hash_sha256 | string | SHA256 hash of the image/binary/file |
| hash_imphash | string | IMPHASH hash of the image/binary/file |
